## 🎯 How to Run the Application

### Option 1: Run Both Servers

**Terminal 1 - Backend Server:**
```bash
cd server
npm run dev
```

**Terminal 2 - Frontend Server:**
```bash
cd client
npm run dev
```

## 🌐 Access the Application

Once both servers are running:

- **Frontend:** http://localhost:5173
- **Backend API:** http://localhost:5000/api
- **Health Check:** http://localhost:5000/api/health

### Role-Based Registration & Themes

**🟢 Customer (Green Theme):**
1. Register at http://localhost:5173/register
2. Access the health-focused dashboard immediately.

**🟡 Pharmacist (Yellow Theme):**
1. Register with professional details (License, NIC).
2. Login as **Admin** to approve the pharmacist registration.
3. Access the clinical dashboard.

**🔵 Admin (Blue Theme):**
1. Login with pre-seeded admin credentials.
2. Oversee all users, pharmacists, and products.

## 🧪 Testing the Workflows

### 1. Prescription-to-Order Flow
- **Customer**: Find a prescription medicine → Upload Rx image → Wait.
- **Pharmacist**: View Rx → Approve → Set Total Price → Confirm.
- **Customer**: View price → Pay → Track Order.

### 2. OTC Shopping
- **Customer**: Add Panadol/Samahan to cart → Checkout → Immediate Order.
